Sobre meu bloco - HTML/CSS

Dificuldades - O css foi minha maior dificuldade, como temia foi necessário utilizar flexbox e infelizmente não tive tanto tempo hábil para aprofundar meus estudos no assunto;

Facilidades - Consegui entender o funcionamento do HTML e assimilei bem o passo a passo para estruturar as tags dentro do documento;

Descobertas - A disciplina como um todo foi uma descoberta, visto que, estudo TI há 11 meses apenas e nunca tinha entrado em contato com esse mundo de códigos e linguagens antes disso, então ver o funcionamento de HTML/CSS/JS tem sido uma grande descoberta, pois venho de outras áreas de atuação profissional como serviço público federal, Direito etc;

Afinidades - Senti afinidade com o desenvolvimento Back-end, mas ainda sou muito limitada, sobretudo na parte de lógica de programação e algoritmos, contudo acho lindo ver os eventos, decorrentes do uso dos métodos e funções que as linguagens de programação possuem.
